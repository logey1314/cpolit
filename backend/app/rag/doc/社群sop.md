社群 SOP
新用户欢迎流程
触发条件
新用户入群后30分钟内自动触发。
欢迎话术模板
欢迎{用户昵称}加入{社群名称}！这里是大家交流{品类}心得的地方。群文件里有{新手指南名称}，可以先看看。有{品类}问题随时在群里问，我们的运营小伙伴会帮你解答~
注意事项
1. 欢迎话术在群内公开发送，不私聊。
2. 用户昵称使用入群时设置的昵称，不使用真实姓名。
3. 新手指南链接有效时才发送，过期链接不发送。
4. 欢迎后24小时内不主动私聊新用户，给用户适应时间。
沉默用户处理
触发条件
用户连续14天无群内互动。
处理方式
1. 不直接私聊促销内容。
2. 在群内发起低打扰话题（如季节性话题、使用心得分享）。
3. 观察用户是否重新参与互动。
4. 连续30天无互动后标记为深度沉默，进入待培育流程。
高意向用户识别
识别信号
1. 连续2天以上在群内询问价格、购买方式或产品对比。
2. 使用"想买""还有货吗""怎么下单"等购买意向关键词。
3. 主动询问活动时间和优惠信息。
处理方式
1. 标记为高意向用户。
2. 建议私域运营私聊跟进。
3. 私聊内容引用当前版本产品资料和活动规则。
4. 私聊前检查触达频控。
活动预热流程
活动前3天
1. 群内发布活动预告（群公告）。
2. 不私聊活动信息，避免打扰。
3. 观察群内用户对活动的反应。
活动当天
1. 群内发布活动正式公告。
2. 对活动关注用户私聊活动邀约。
3. 活动邀约内容必须过品牌合规审核。
4. 私聊前检查触达频控。
活动结束后
1. 群内发布活动感谢公告。
2. 对已购用户发送使用提醒。
3. 对未购的高意向用户保持跟进，但不立即再次私聊。
