品牌规范
品牌语气
总体调性
品牌语气定位为"专业、温和、真诚"。运营人员在生成私聊话术、社群话题和朋友圈文案时，应保持以下语气特征：
1. 专业但不生硬：用易懂的语言解释产品功效，避免过度使用专业术语。
2. 温暖但不谄媚：关心用户需求，但不使用"亲""宝贝"等过度亲昵称呼。
3. 真诚但不夸大：如实描述产品效果，不使用绝对化表述。

禁用词表
绝对禁止
● 全网最低价、全网最便宜
● 保证有效、百分百有效
● 永久修复、彻底根治
● 限时一天、最后机会（除非真实活动规则）
● 随便退、无条件退款
需人工确认
● 限时、限量（需核实活动规则）
● 全网首发（需确认是否真实首发）
● 独家、唯一（需确认是否真实独家）
● 手速要快、错过没有了（催促性表达，需人工确认）
权益边界
1. 会员专享价仅对会员用户提及，不得对非会员承诺会员价。
2. 限量产品必须说明限量数量和售完即止。
3. 满减规则必须说明适用范围和不可叠加项。
4. 赠品必须说明数量有限和先到先得。
承诺边界
1. 不得承诺超出退换货政策的退款条件。
2. 不得承诺产品效果（如"用了一定会好"）。
3. 不得承诺发货时间（除非有明确物流承诺）。
4. 不得承诺价格不会变动。
竞品表达
1. 不得在内容中点名竞品品牌。
2. 不得做负面对比（如"比XX品牌好"）。
3. 可以做客观品类对比（如"相比传统面霜，这款质地更轻薄"）。
隐私保护
1. 不得在群内公开用户购买记录。
2. 不得在群内透露其他用户的个人信息。
3. 私聊内容不得在群内转发。
